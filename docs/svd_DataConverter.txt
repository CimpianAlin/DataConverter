1.2  System overview
The DataConverter component is used to convert between BULKIO data types in REDHAWK.  With proper configuration, the DataConverter component can convert between any of the following data types; Char, Octet, Unsigned Short, Short, and Float.  The DataConverter component is also capable of converting real data into complex data, and similarly complex data to real data.
